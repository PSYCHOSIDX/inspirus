Code Trade
Rules
  • The students participating should be from the same institution (Interdepartmental teams allowed).
  • A PC with Microsoft Visual Studio 2010 and NetBeans IDE 8.2 installed for C/C++/Java will be provided to each team.
  • Maximum 15 teams will be allowed to register after which registrations will be closed.
  • Round 1: - The participants have to frame the logic for the given problem statements, code it and submit whatever has been completed within 30 minutes. Bonus point if participants complete the solution before time.
  • Break of 15 minutes.
  • Round 2: - Codes will be swapped between partners. They have to complete the second half of each other‚Äôs incomplete code within 30 minutes. Bonus point if participants complete given statements before time.
  • Note: both the team mates will be coding simultaneously and codes will be swapped at the end of round 1.
  • Total Time Duration: 60 minutes coding + 15minutes break= 1hr 15min
  • Ties will be broken by the total time for each user in ascending order of time. A program will be considered wrong if it fails to work on one or more of the test cases. A correct program will be awarded full points, points will be deducted for incomplete code.
  • The Contest Committee will tally the scores for each team and publish a complete list. The teams with the highest scores will be declared the winners. The decision of the Committee will be final and abiding.
  • All the team members are requested to follow all the Safety Protocols during the entirety of the event.
